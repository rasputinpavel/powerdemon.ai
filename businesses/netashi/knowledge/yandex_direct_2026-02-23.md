## 1. Параметры кампании

- **Тип:** Поиск + РСЯ (Рекламная сеть Яндекса)
- **География:** Россия (все регионы), Казахстан, Беларусь
- **Посадочная:** Tilda-сайт (после публикации)
- **Цель:** Заявки через форму на сайте + клик по WhatsApp
- **Рекомендуемый бюджет:** 500-1000₽/день на старте (15-30 тыс₽/мес)

## 4. Настройки кампании

```markdown

- Стратегия: Оптимизация конверсий (после набора 10+ конверсий) / Ручное управление ставками (на старте)
- Режим показа: стандартный
- Корректировки ставок:
  - Мобильные: +20% (мамы ищут с телефона)
  - Возраст 25-34: +30%
  - Возраст 35-44: +15%
  - Пол женский: +20%
- Расписание: круглосуточно (разные часовые пояса)
- Дневной лимит: 500-1000₽
```

## 7. Оптимизация (после запуска)

### Неделя 1-2: Сбор данных
- Не трогать, наблюдать за CTR и ценой клика
- Отключить ключевые слова с CTR < 1%

### Неделя 3-4: Оптимизация
- Отключить неэффективные ключи
- Добавить минус-слова из поисковых запросов
- Тестировать новые заголовки
- Перераспределить бюджет на конвертящие группы
