# Assets — Креативные ресурсы

Папка для хранения визуальных и медиа-ресурсов бизнеса.

## Структура
- `photos/` — оригинальные фотографии продуктов и услуг
- `processed/` — обработанные изображения (с фоном, брендингом)
- `brand/` — логотипы, фирменные цвета, шрифты
- `references/` — референсы стиля, примеры конкурентов

## Использование
Агент использует ресурсы из этой папки при создании контента:
- Фото из `photos/` обрабатываются и используются в постах
- Элементы из `brand/` применяются для единого стиля
- Файлы из `references/` задают визуальное направление

Просто кидай сырые фото сюда — агент обработает и использует.
