# Выход на новый рынок

## Вход

- Прочитай `context.md` в папке бизнеса
- Прочитай все файлы в `knowledge/` (исследования, ICP, конкуренты)

## Шаг 1: Определить параметры

Уточни у оператора:
1. **Какой рынок?** (география, сегмент, или и то и другое)
2. **Почему этот рынок?** (гипотеза, возможность)
3. **Что адаптируем?** — тот же продукт или новый?
4. **Бюджет на вход** — сколько готовы вложить?
5. **Таймлайн** — когда нужны первые результаты?

## Шаг 2: Исследование нового рынка

Выполни мини-версию `/market_research` для целевого рынка:
1. Размер и потенциал
2. Ключевые игроки на этом рынке
3. Регуляторные особенности
4. Культурные особенности (если другая география)
5. Барьеры входа

## Шаг 4: Определить каналы входа

Предложи 3-5 каналов с оценкой:
```markdown
| Канал | Стоимость | Скорость | Потенциал | Приоритет |
```

## Шаг 5: Составить план действий

```markdown

### Фаза 1: Валидация (1-2 недели)
- [ ] ...

### Фаза 2: Тестовый запуск (2-4 недели)
- [ ] ...

### Фаза 3: Масштабирование (если Фаза 2 успешна)
- [ ] ...
```

## Шаг 6: Сохранить результат

Файл: `businesses/{business_name}/projects/{project}/assets/market_entry_plan_{YYYY-MM-DD}.md`

Обнови `log.md` текущего проекта.
