# Запуск рекламной кампании в Яндексе

## Вход

- Прочитай `context.md` в папке бизнеса (продукт, ICP, география)
- Прочитай `brief.md` текущего проекта (цель, бюджет, KPI)
- Если есть `knowledge/icp_*.md` — используй для таргетинга

## Шаг 1: Определить параметры кампании

Уточни у оператора:
1. **Тип кампании:** поиск / РСЯ / ретаргетинг
2. **География:** регионы показа
3. **Бюджет:** дневной и общий
4. **Посадочная страница:** URL куда ведём
5. **Цель:** заявки / звонки / покупки / трафик

## Шаг 2: Собрать семантическое ядро

Через веб-поиск и анализ ниши:
1. Определи **основные ключевые запросы** (15-30 штук)
2. Сгруппируй по **тематическим группам** (3-7 групп)
3. Определи **минус-слова** (10-20 штук)
4. Оцени **частотность** (если возможно через Wordstat)

```markdown
### Группа: {название}
Ключевые слова:
- {запрос 1}
- {запрос 2}

Минус-слова:
- {слово 1}
```

## Шаг 3: Написать объявления

Для каждой группы ключевых слов напиши:
- **Заголовок 1** (до 56 символов) — содержит ключевое слово
- **Заголовок 2** (до 30 символов) — УТП или призыв
- **Текст объявления** (до 81 символа) — выгода + CTA
- **Быстрые ссылки** (4 штуки)
- **Уточнения** (4 штуки)

## Шаг 4: Рекомендации по настройкам

```markdown
## Настройки кампании
- Стратегия: {рекомендация}
- Корректировки ставок: {что повысить/понизить}
- Расписание показов: {когда показывать}
- Устройства: {мобильные/десктоп}
```

## Шаг 5: Составить чек-лист запуска

```markdown
## Чек-лист перед запуском
- [ ] Проверить посадочную страницу (грузится, формы работают)
- [ ] Установить Яндекс.Метрику с целями
- [ ] Загрузить кампанию в Директ
- [ ] Проверить объявления на модерации
- [ ] Установить дневной лимит бюджета
- [ ] Настроить уведомления о расходе
```

## Шаг 6: Сохранить результат

Файл: `businesses/{business_name}/projects/{project}/assets/yandex_campaign_{YYYY-MM-DD}.md`

Обнови `log.md` текущего проекта.
