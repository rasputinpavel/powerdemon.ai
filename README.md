# PowerDemon.AI — AI-first маркетинговое агентство

Система для управления маркетинговыми проектами нескольких бизнесов через AI-агентов.

## Как устроена система

```
Antygravity/
├── .agent/workflows/     ← Скиллы (переиспользуемые воркфлоу)
├── _templates/           ← Шаблоны документов
└── businesses/           ← Бизнесы (изолированные контексты)
    └── {name}/
        ├── context.md    ← Контекст бизнеса
        ├── projects/     ← Проекты
        └── knowledge/    ← Накопленные знания
```

## Принципы

1. **Изоляция бизнесов** — каждый бизнес в своей папке, контексты не пересекаются
2. **Переиспользуемые скиллы** — один воркфлоу работает для любого бизнеса
3. **Antigravity = оркестратор** — управление через диалог, скиллы вызываются по `/command`

## Скиллы (доступные команды)

| Команда | Описание |
|---|---|
| `/onboard_client` | Онбординг нового клиента |
| `/new_project` | Запуск нового проекта |
| `/market_research` | Deep Research рынка |
| `/competitor_analysis` | Анализ конкурентов |
| `/icp_analysis` | ICP + сегментация |
| `/traction_map` | Трекшн-карта экспериментов |
| `/ad_campaign_yandex` | Запуск рекламы в Яндексе |
| `/ad_campaign_google` | Запуск рекламы в Google Ads |
| `/new_market_entry` | Выход на новый рынок |
| `/weekly_report` | Еженедельный отчёт |

## Быстрый старт

```
"Давай поработаем с {business_name}"  → переключение на бизнес
"/onboard_client"                     → создание нового бизнеса
"/new_project"                        → запуск проекта
```

## Масштабирование (Этап 2)

Когда задач станет больше, Antigravity начнёт делегировать тяжёлые скиллы в Claude Code subagents:
```bash
claude -p "Контекст: businesses/{name}/context.md. Задача: ..." 
```
Формат скиллов уже готов к этому — каждый воркфлоу самодостаточен.
